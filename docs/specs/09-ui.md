# Spec 09 – UI standards

- **Theme:** CSS variables (shadcn-style); optional dark mode via `dark` class. Admin + Station use the same UI stack: shadcn/ui + Tailwind + Lucide (admin); station apps and Kiosk use `@traceability/ui` + Tailwind.
- **Admin:** Desktop; Shell; list/detail/form patterns. See [../ui/guidelines.md](../ui/guidelines.md).
- **Station:** Kiosk/touch; full-screen flows; large targets; minimal steps.
- **State:** Server = React Query; UI = useState/Context. Types from SDK.
