# Spec 19 – Revision and variant governance

- **Revision:** Immutable once ACTIVE (Gate A). Draft → Active → Inactive. No edit of active revision.
- **Variant:** Optional; e.g. WITH_SHROUD / NO_SHROUD for Marlin. Assignable only before first assembly step (Gate B).
- **Readiness:** REVISION_NOT_READY enforced; config and BOM/routing must be complete for active revision.
