# 06 Event Catalog (Authoritative)

## Event principles

- All events have unique event_id (UUID) generated by client.
- Server is idempotent: same event_id => no duplicate side effects.
- Events include:
  - unit_id (primary target)
  - machine_id
  - created_at_device
  - payload (JSON)
  - operator_user_id resolved server-side (JWT or device operator session)

## Core event list

### Dispatch (Production ↔ Jigging)

- DISPATCH_CREATED (production)
- DISPATCH_CONFIRMED (jigging)
- DISPATCH_RETURNED (production)

### Plate flow

- PLATE_LOADED (jigging creates PLATE_120)
- WASH1_START (optional)
- WASH1_END (plate washed)
- BONDING_PLATE_SCANNED (plate consumed into bonding)

### Magnet flow

- MAGNET_PREPARED (jigging)
- BONDING_MAGNET_SCANNED (consume 120 from pack)
- MAGNET_CARD_RETURNED (to jigging)

### Component jigs

- JIG_LOADED (jigging creates jig unit)
- WASH2_START
- WASH2_END (jig status WASH2_COMPLETED)
- JIG_RETURNED (back to jigging staging)

### Bonding

- BONDING_START (input plate+magnet)
- BONDING_END (creates ASSY_120 and links input units)

### Magnetize / Flux

- MAGNETIZE_DONE
- FLUX_PASS
- FLUX_FAIL (go to HOLD)

### Assembly binding

- ASSY_BIND_COMPONENTS
  payload:
  - assy_unit_id
  - selected_variant (if not assigned yet)
  - component_jig_unit_ids: { PIN430, PIN300, SHROUD(optional), CRASH }
  - line_code (from machine config)

### Assembly steps (consumption happens here)

- PRESS_FIT_PIN430_DONE (consume qty 120 from PIN430_JIG)
- PRESS_FIT_PIN300_DONE (consume qty 120 from PIN300_JIG)
- PRESS_FIT_SHROUD_DONE (only WITH_SHROUD)
- CRASH_STOP_DONE (consume qty 120 from CRASH_JIG)
- IONIZER_DONE
- FVMI_PASS (creates trays)
- FVMI_FAIL (HOLD)

### Labeling

- LABEL_GENERATE_REQUEST (online only)
- LABELS_GENERATED (creates 6 labels + attach to trays)

### Split / Packing / FG

- SPLIT_GROUP_CREATED (2 groups of 60)
- OUTER_PACKED (3 trays per outer)
- FG_PALLET_MAPPED

## Standard validation outcomes

- INVALID_STATE_TRANSITION
- MISSING_REQUIRED_COMPONENT
- COMPONENT_NOT_WASHED
- INSUFFICIENT_QTY_REMAINING
- VARIANT_MISMATCH
- LINE_NOT_CAPABLE_FOR_VARIANT
- OFFLINE_SERIAL_NOT_ALLOWED
